# 📈 Zerodha – Stock Trading Platform

A full-stack stock trading platform that enables users to buy/sell stocks in real-time with live market data, portfolio tracking, and interactive analytics.

## 🌟 Key Features

- ✅ **Real-time Market Data** - Live stock prices via WebSockets
- ✅ **Buy/Sell Orders** - Instant trade execution
- ✅ **Portfolio Management** - Track all holdings and investments
- ✅ **P&L Analytics** - Automated profit/loss calculations
- ✅ **Interactive Dashboards** - Visual representation of portfolio
- ✅ **Order History** - Complete transaction records
- ✅ **Real-time Notifications** - Live updates on price movements
- ✅ **User Authentication** - Secure login and account management
# 🛠️ Tech Stack

| Component | Technology |
|-----------|------------|
| **Frontend** | React.js, JavaScript, HTML, CSS |
| **Backend** | Node.js, Express.js |
| **Database** | MongoDB |
| **Real-time** | WebSockets, socket.io |
| **APIs** | REST APIs, Market Data APIs |
| **Authentication** | JWT |
